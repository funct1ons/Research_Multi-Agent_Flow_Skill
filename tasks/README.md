# Tasks

This directory stores task cards for work on this repository and provides a reusable template for downstream research projects.

Use one task card for one bounded task. A good task card has:

- one goal
- a branch name
- allowed files
- forbidden files
- acceptance criteria
- validation commands
- review checklist
- handoff section

Use [task-card-template.md](task-card-template.md) when creating a new task.
